Descarga las dependecias del proyecto
```
npm install
```

Monta en modo desarrollo mi app para probarla
```
npm run dev
```

Construir la aplicación
```
npm run build
```

Instalar una dependencia en concreto
```
npm i *nombre de la dependencia*
```

Deploy a github
```
npm run deploy
```
